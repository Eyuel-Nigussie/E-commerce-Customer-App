## üöÄ Key Features
- **Dynamic Product Catalog:** Real-time data fetching with categorized views.
- **Reactive State Management:** Seamless UI updates using **GetX (Obx)** for cart and wishlist.
- **Secure Authentication:** Integrated with **Firebase Auth** (Email/Password & Google).
- **Advanced Navigation:** Custom routing and transition animations.
- **Cart System:** Fully functional cart with quantity management and price calculations.
- **Search & Filter:** Optimized search bar for finding products instantly.
